# モンスターリーグ ファクトリー

ブラウザだけで遊べる、ターン制モンスターバトルゲームです。
インストール不要、ダウンロード不要。このページを開くだけですぐに遊べます。

## 遊び方

1. タイトル画面で名前を入力して「ちょうせん スタート」をタップ
2. ランダムに提示される6体の中から、3体をレンタルして出撃
3. 勝利すると、相手のモンスター1体と自分のモンスター1体を交換できます
4. 連勝を重ねるほど、相手はどんどん手強くなっていきます

## 特徴

- オリジナルモンスターが100体以上登場
- タイプ相性・状態異常・能力変化・交代技など、本格的なバトルシステム
- 勝利のたびに成長する「撃破ボーナス」システム
- 隠しモンスターや専用技を持つ強敵との対決
- 出会ったモンスターを記録する図鑑機能
- 実績システムで遊びごたえをさらに追加
- 全国のプレイヤーと連勝数を競えるランキング機能

## 技術構成

HTML・CSS・JavaScriptのみで動作するシングルファイル構成です。
ランキング機能にはFirebase Realtime Databaseを利用しています。
